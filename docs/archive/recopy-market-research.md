## 一、市场概况

剪贴板管理工具是一个成熟但持续演进的细分市场。用户核心痛点始终未变：**操作系统原生剪贴板只能保存最近一条记录，频繁复制粘贴时极易丢失重要内容。**

### 市场趋势

1. **Apple 入场**：macOS Tahoe（26）在 Spotlight 中内置剪贴板历史，但功能基础。说明 Apple 承认了需求但留下了大量空白。
2. **Windows 同样内置**：Win+V 剪贴板历史，仅 25 条上限，功能有限。PowerToys Advanced Paste 在 AI+剪贴板方向探路。
3. **AI 能力尚未渗透**：市场上几乎没有剪贴板工具集成 AI（智能分类、摘要、语义搜索），潜在差异化方向。
4. **跨平台需求强烈**：多数用户同时使用 Mac + Windows，但优秀的跨平台方案很少。
5. **订阅疲劳**：用户对小工具的订阅制普遍反感，买断制或免费更受欢迎。

## 二、核心风险：系统原生剪贴板历史的威胁分析

> **这是产品立项必须回答的生死问题：如果系统原生支持足够好，第三方产品必死。**

### 2.1 macOS Tahoe（26）原生剪贴板历史

macOS 26 Tahoe（2025.9 正式发布）首次在 Spotlight 中内置剪贴板历史。

**功能清单：**
- 入口：Spotlight → Clipboard（Cmd+Space 后 Cmd+4）
- 支持：纯文本、图片、文件、URL
- 搜索：Spotlight 内搜索
- 操作：双击粘贴、右键删除/复制
- 去重：重复复制移到顶部
- 保留时长：30 分钟 / 8 小时 / 7 天
- 默认关闭

**致命缺陷（截至 26.3）：**

| 缺陷 | 严重程度 | 说明 |
|------|---------|------|
| **仅保存纯文本** | 🔴 致命 | 复制富文本时格式全部丢失 |
| **最长仅 7 天** | 🔴 致命 | 无法作为长期片段库 |
| **无 Pin/收藏** | 🔴 致命 | 所有内容都会过期消失 |
| **约 20 条上限** | 🟡 重要 | 远不如专业工具的无限历史 |
| **入口笨拙** | 🟡 重要 | Cmd+Space → Cmd+4 两步 |
| **安全隐患** | 🔴 致命 | 屏幕共享时内容明文暴露 |

**社区评价**：MacRumors 用户称其 "barely an MVP"。

**结论：macOS 原生 = 入门级能力，不构成替代威胁，但会教育市场。**

### 2.2 Windows 11 原生剪贴板历史

**功能清单：**
- 入口：Win+V（一步直达）
- 支持：文本、HTML、Bitmap
- 条目上限：25 条（硬编码）
- Pin（固定）：✅
- 云同步：✅（仅 Windows 设备间）
- 重启清除未固定条目

**致命缺陷：**

| 缺陷 | 严重程度 |
|------|---------|
| **仅 25 条** | 🔴 致命 |
| **无搜索** | 🔴 致命 |
| **重启清除** | 🟡 重要 |
| **不支持文件** | 🟡 重要 |

### 2.3 PowerToys Advanced Paste

Microsoft PowerToys 的 Advanced Paste（Win+Shift+V）：
- 格式转换、本地 OCR、AI 能力（可选本地/云端模型）
- **但 ≠ 剪贴板管理器**：没有历史记录、搜索、收藏

### 2.4 威胁评估矩阵

| 维度 | macOS 原生 | Windows 原生 | Recopy 目标 |
|------|-----------|-------------|-----------|
| 历史条数 | ~20, 最长7天 | 25条, 重启清除 | 无限, 永久 |
| 搜索 | ✅ 基础 | ❌ | ✅ FTS5 全文 |
| 富文本 | ❌ 仅纯文本 | ✅ HTML | ✅ 完整格式 |
| Pin/收藏 | ❌ | ✅ | ✅ |
| 跨平台 | ❌ | ❌ | ✅ Mac+Win |
| 视觉化UI | ⭐ | ⭐ | ⭐⭐⭐ |

### 2.5 关键结论

1. **短期（1-2年）：不构成威胁，反而利好** — 原生功能教育市场
2. **安全区域**：跨平台统一体验、视觉化卡片流、无限历史、收藏管理 — 系统原生永远不会做
3. **Recopy 定位**：利用原生功能教育市场后的"升级需求"

## 五、Recopy 产品定位与竞争分析

### 定位

> **Recopy = 对标 Paste 体验的跨平台剪贴板管理工具**
>
> 在 Mac 和 Windows 上提供统一的视觉化卡片体验，支持纯文本/富文本/图片/文件的完整记录、全文搜索和收藏管理。

### 为什么这个定位有机会

目前没有一款产品同时满足 "Paste 级体验 + 跨平台"：

| 产品 | 体验 | 跨平台 |
|------|------|--------|
| Paste | ⭐⭐⭐ 标杆 | ❌ Apple only |
| Ditto | ⭐ UI 老旧 | ❌ Windows only |
| EcoPaste | ⭐⭐ 尚可 | ✅ 全平台 |
| **Recopy** | **⭐⭐⭐ 对标 Paste** | **✅ Mac+Win** |

**空白地带 = Recopy 的机会**：体验最好的锁死 Apple 生态，跨平台的体验粗糙。

### 相对原生的核心优势

| 原生做不到 | Recopy 做到 |
|-----------|-----------|
| macOS 只存纯文本 | 富文本 + 图片 + 文件 |
| macOS 最长 7 天 / Windows 25 条 | 无限历史 |
| macOS 无 Pin | Pin 收藏 |
| Windows 无搜索 | FTS5 全文搜索 |
| 不跨平台 | Mac + Windows 统一体验 |
| 入口笨拙 | 单快捷键直达 |

### 核心竞争逻辑

```
Mac 用户：  "系统原生只存纯文本，Paste 要 $30/年"
            → Recopy 免费，富文本+图片+文件，且换 Windows 也能用

Win 用户：  "Win+V 只有 25 条还没搜索"
            → Recopy 无限历史 + 搜索 + 现代 UI

双系统用户："Mac 上用 Paste，Windows 上用 Ditto，体验割裂"
            → Recopy 一套工具两个平台
```
